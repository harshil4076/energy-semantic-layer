# metric_daily

**Dataset:** `main`

## Description

Daily roll-up of household energy consumption. The primary view for cost and consumption questions.
Each row is one calendar day. Consumption is in kWh; cost is in EUR.

Peak hours are 06:00–21:59 (EDF Bleu definition). Off-peak is 22:00–05:59.
Cost is estimated using a flat tariff of €0.18/kWh — excludes taxes and standing charges.
